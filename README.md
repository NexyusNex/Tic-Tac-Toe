# Tic-Tac-Toe

![App-screenshot](https://github.com/NexyusNex/Tic-Tac-Toe/blob/main/screenshot.png?raw=true)

The classic Tic Tac Toe game where you can choose the names of the two players and play.

## Features

- Change player names
- Fun for 2 players
- Reset the board

## Built using

- HTML
- CSS
- Javascript
